# MTG (WIP)

> MTG utilities for node.

## CLI

```
mtg info|i <card>
    curve <decklist>
    -h|--help|help [<command>]
    -v|--version|version
```

## Docs

Use `npm run docs` to generate docs. This will run the `./scripts/docs.js`
script.

## Testing

We're using mocha for our tests:

`npm test`
